# Rowen Infante Portfolio Design

## Goal

Create a personal portfolio website for Rowen Infante inspired by the clean, compact, profile-forward style of Bryl Lim's portfolio while keeping the content and identity original.

## Direction

The portfolio presents Rowen as a hybrid SAP, AI, and web development professional. The story is: manufacturing engineering background, process engineering experience, current SAP industry work at Accenture, and a growing practice in web development and AI-generated creative work.

## Sections

- Hero: Rowen Infante, SAP BTP Administrator, SAP and AI-focused builder.
- About: concise career story from manufacturing engineering to SAP BTP, AI, and web development.
- SAP BTP Skills: destinations, roles, identity, CTMS, CI/CD, SAP Build Work Zone, SAP HANA, SAP Generative AI.
- Projects: Obsidian skincare ecommerce, personal website under development, AI video creation.
- Certifications: SAP BTP Administrator, SAP Generative AI Developer, Google/Coursera data foundations.
- Experience: Accenture, SFA Semicon Philippines, Mapua University.
- Video Gallery: local AI-generated videos supplied by Rowen.
- Contact: LinkedIn, GitHub, and email placeholders.

## Visual Style

The UI uses a black-and-white personal index layout inspired by Bryl Lim's style: fixed left sidebar, dotted background, compact monospace navigation, centered profile hero, thin dividers, monochrome cards, and restrained hover/click effects. The hero includes a profile-style visual, simple metrics, magnetic links, animated skill chips, and a command-card style prompt.

## Interaction

- Hero buttons respond to pointer movement with a magnetic hover effect.
- Cards lift and rotate subtly on hover.
- Sections reveal as the user scrolls.
- Skill chips have staggered entrance timing.
- Hero command card can be clicked to cycle short personal prompts.

## Scope

The initial version is a standalone static site using HTML, CSS, and JavaScript. It does not require package installation, a database, or a build process.
